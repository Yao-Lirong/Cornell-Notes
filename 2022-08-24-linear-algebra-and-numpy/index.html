Vector Space¶

a vector space over the real numbers is a set VVV together with two binary operations +++ (mapping V×VV \times VV×V to VVV) and ⋅\cdot⋅ (mapping R×V\R \times VR×V to VVV) satisfying the following axioms for any x,y,z∈Vx, y, z \in Vx,y,z∈V and a,b∈Ra, b \in \Ra,b∈R

  • closure: x+y∈Vx + y \in Vx+y∈V and a⋅x=ax∈Va \cdot x = ax \in Va⋅x=ax∈V
  • associativity of addition: (x+y)+z=x+(y+z)(x + y) + z = x + (y + z)(x+y)+z=x+(y+z)
  • transitivity of addition: x+y=y+xx + y = y + xx+y=y+x
  • negation: there exists a (−x)(-x)(−x) such that x+(−x)=0x + (-x) = 0x+(−x)=0
  • zero element: 0∈V0 \in V0∈V such that 0+x=x+0=x0 + x = x + 0 = x0+x=x+0=x
  • associativity of scalar multiplication: a(bx)=b(ax)=(ab)xa(bx) = b(ax) = (ab)xa(bx)=b(ax)=(ab)x
  • multiplication by one: 1v=v1v = v1v=v
  • distributivity: a(x+y)=ax+aya(x+y) = ax + aya(x+y)=ax+ay and (a+b)x=ax+bx(a+b)x = ax + bx(a+b)x=ax+bx

Linear Map¶

We say a function FFF from a vector space UUU to a vector space VVV is a linear map if for any x,y∈Ux, y \in Ux,y∈U and any a∈Ra \in \Ra∈R,

F(ax+y)=aF(x)+F(y)F(ax + y) = a F(x) + F(y)F(ax+y)=aF(x)+F(y)

Matrix¶

We have basically introduced the “matrix” in above. We call this two-dimensional-array representation of a linear map a matrix.

We use multiplication to denote the effect of a matrix operating on a vector (this is equivalent to applying a multilinear map as a function). E.g. if FFF is the multilinear map corresponding to matrix AAA:

y=F(x)  ⟺  y=AxF:Rm→Rn  ⟺  A∈Rn×my = F(x) \iff y = Ax \\ F:\R^m \to \R^n \iff A \in \R^{n \times m}y=F(x)⟺y=AxF:Rm→Rn⟺A∈Rn×m
(3)

We can add two matrices, scale a matrix by a scalar, and these two operations follow the “axioms” described above for a vector space. This means that the set of matrices Rn×m\R^{n \times m}Rn×m is itself a vector space. (This was covered in MATH3360)

Broadcasting¶

When operate on vectors / matrices, numpy broadcasts by acting as if there are infinite 1×1\times1× to the left of dimension:

if dimension of array not same:
    append newaxis to the lesser one until same
    if on a specific axis, one array has dim 1 and the other has > 1:
        broadcast that 1 axis to match dimension
    else:
        fail

However, it is good practice for you to always manually add dimension before broadcasting.

x = numpy.array([2,3])
x1 = x[numpy.newaxis,:]
