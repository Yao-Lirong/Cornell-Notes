Well-defined kernels¶

We can build kernels by recursively combining one or more of the following rules:

  1. k(x,z)=ck1(x,z)\mathsf{k}(\mathbf{x}, \mathbf{z})=c\mathsf{k_1}(\mathbf{x},\mathbf{z})k(x,z)=ck1​(x,z)
  2. k(x,z)=k1(x,z)+k2(x,z)\mathsf{k}(\mathbf{x}, \mathbf{z})=\mathsf{k_1}(\mathbf{x},\mathbf{z})+\mathsf{k_2}(\mathbf{x},\mathbf{z})k(x,z)=k1​(x,z)+k2​(x,z)
  3. k(x,z)=g(k(x,z))\mathsf{k}(\mathbf{x}, \mathbf{z})=g(\mathsf{k}(\mathbf{x},\mathbf{z}))k(x,z)=g(k(x,z))
  4. k(x,z)=k1(x,z)k2(x,z)\mathsf{k}(\mathbf{x}, \mathbf{z})=\mathsf{k_1}(\mathbf{x},\mathbf{z})\mathsf{k_2}(\mathbf{x},\mathbf{z})k(x,z)=k1​(x,z)k2​(x,z)
  5. k(x,z)=f(x)k1(x,z)f(z)\mathsf{k}(\mathbf{x}, \mathbf{z})=f(\mathbf{x})\mathsf{k_1}(\mathbf{x},\mathbf{z})f(\mathbf{z})k(x,z)=f(x)k1​(x,z)f(z)
  6. k(x,z)=ek1(x,z)\mathsf{k}(\mathbf{x}, \mathbf{z})=e^{\mathsf{k_1}(\mathbf{x},\mathbf{z})}k(x,z)=ek1​(x,z)

where k1,k2k_1,k_2k1​,k2​ are well-defined kernels, c≥0c\geq 0c≥0, ggg is a polynomial function with positive coefficients, fff is any function and A⪰0\mathbf{A}\succeq 0A⪰0 is positive semi-definite.

Example: Algorithm with Squared Loss¶

Recap¶

linear regression minimizes the following squared loss:

min⁡w∑i=1n(w⊤xi−yi)2\min_\mathbf{w} \sum_{i=1}^{n} (\mathbf{w}^\top \mathbf{x}_i -y_i)^2wmin​i=1∑n​(w⊤xi​−yi​)2
(1)

The solution of OLS can be written in closed form:

w=(XX⊤)−1Xy\mathbf{w}=(\mathbf{X}\mathbf{X}^\top)^{-1} \mathbf{X} \mathbf{y}w=(XX⊤)−1Xy

Note here our x\mathbf xx has already gone through the transformation ϕ\phiϕ into the feature space, so x=ϕ(xoriginal  data)\mathbf x = \phi(\mathbf x_{original \;data})x=ϕ(xoriginaldata​). Therefore, the conclusion we make here generalizes to all kinds of kernel with k(xog  i,xog  j)=ϕ(xog  i)⊤ϕ(xog  j)\mathsf{k}(\mathbf{x}_{og \;i},\mathbf{x}_{og \;j}) =\phi(\mathbf{x}_{og \;i})^\top \phi(\mathbf{x}_{og \;j}) k(xogi​,xogj​)=ϕ(xogi​)⊤ϕ(xogj​) and K=XTX\mathsf K = \mathbf X ^T \mathbf XK=XTX.
