Boosting Reduces Bias¶

Scenario: Hypothesis class H\mathbb{H}H, whose set of classifiers has large bias and the training error is high (e.g. CART trees with very limited depth.)

Famous question: Can weak learners (hhh) be combined to generate a strong learner with low bias?

Answer: Yes!

Case study #1: Gradient Boosted Regression Tree(GBRT)¶

Background and Setting¶

  • Classification (yi∈{+1,−1} y_i \in \{+1,-1\} yi​∈{+1,−1}) or regression (yi∈Rky_i\in\mathcal{R}^kyi​∈Rk)
  • Weak learners,h∈Hh \in \mathbb{H}h∈H, are not too deep fixed-depth (e.g. depth=4) regression trees .
  • Step size α\alphaα is fixed to a small constant (hyper-parameter).
  • Loss function: Any differentiable convex loss that decomposes over the samples L(H)=∑i=1nℓ(H(xi))\mathcal{L}(H)=\sum_{i=1}^{n} \ell(H(\mathbf{x}_i))L(H)=∑i=1n​ℓ(H(xi​))

Goal¶

We want to find a tree hhh that maximizes h=argminh∈H∑i=1nrih(xi)h = \textrm{argmin}_{h \in \mathbb{H}} \sum_{i = 1}^{n} r_i h(\mathbf{x}_i)h=argminh∈H​∑i=1n​ri​h(xi​) where ri=∂ℓ∂H(xi)r_i = \frac{\partial \ell}{\partial H(\mathbf{x}_i)}ri​=∂H(xi​)∂ℓ​.

Assumptions¶

  1. First, we assume that ∑i=1nh2(xi)\sum_{i = 1}^{n} h^2(\mathbf{x}_i)∑i=1n​h2(xi​) = constant. So we are essentially fixing the vector hhh in ∑i=1nh(xi)ri\sum_{i=1}^n h(\mathbf{x}_i)r_i∑i=1n​h(xi​)ri​ to lie on a circle, and we are only concerned with its direction but not its length.
  2. Define the negative gradient as ti=−ri=−∂ℓ∂H(xi)t_i = -r_i = -\frac{\partial \ell}{\partial H(\mathbf{x}_i)}ti​=−ri​=−∂H(xi​)∂ℓ​.

Therefore, we translate the original problem about finding a regression tree that minimizes an arbitrary loss function to this problem of finding a regression tree that minimizes the squared loss with our new “label” ttt and we are fitting ttt now.

If the loss function ℓ\ellℓ is the squared loss, i.e. ℓ(H)=12∑i=1n(H(xi)−yi)2\ell(H)=\frac{1}{2}\sum_{i=1}^n (H(\mathbf{x}_i)-y_i)^2ℓ(H)=21​∑i=1n​(H(xi​)−yi​)2, there is a special meaning of this ttt: it is easy to show that

which is simply the error between our current prediction and the correct label, so this newly added model is just fitting the error current ensemble model has. However, it is important to keep in mind that you can use any other differentiable and convex loss function ℓ\ellℓ and the solution for your next weak learner hhh will always be the regression tree minimizing the squared loss.

Finding the stepsize α\alphaα¶

In the previous example, GBRT, we set the stepsize α\alphaα to be a small constant. As it turns out, in the AdaBoost setting we can find the optimal stepsize (i.e. the one that minimizes ℓ\ellℓ the most) in closed form every time we take a “gradient” step.

If we take the derivative of loss function ℓ\ellℓ with respect to α\alphaα, we will actually find out that there is a nice closed form for α\alphaα:

α=12ln⁡1−ϵϵ\alpha = \frac{1}{2}\ln \frac{1-\epsilon}{\epsilon}α=21​lnϵ1−ϵ​
(13)

It is unusual that we can find the optimal step-size in such a simple closed form. One consequence is that AdaBoost converges extremely fast.

The inner loop can terminate as the error ϵ=12\epsilon=\frac{1}{2}ϵ=21​, and in most cases it will converge to 12\frac{1}{2}21​ over time. Further analysis¶

We can in fact show that the training loss is decreasing exponentially! Even further, we can show that after O(log⁡(n))O(\log(n))O(log(n)) iterations your training error must be zero. In practice it often makes sense to keep boosting even after you make no more mistakes on the training set.
